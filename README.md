# Mahendra Seervi - 3D Portfolio

A responsive static portfolio for Mahendra Seervi, a Computer Science engineering student at IIIT Bhagalpur and full-stack developer.

## Highlights

- Three.js animated hero with orbiting skill nodes, constellation lines, particles, and pointer parallax
- Dark/light theme toggle with saved preference
- Mobile navigation, scroll reveals, and animated impact counters
- Resume-driven sections for about, education, skills, freelance experience, projects, achievements, and contact
- Direct email, phone, and resume download actions

## Files

- `index.html` - Portfolio markup and content
- `style.css` - Responsive visual system and layout
- `script.js` - Theme, navigation, scroll animation, counters, and Three.js scene
- `Resume_Mahendra (30).pdf` - Downloadable resume

Open `index.html` in a browser to view the site.
